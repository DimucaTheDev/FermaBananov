# FermaBananov
Наипростейшая ферма бананов для банана кликера

# Как работает?
1. Запускает игру через STEAM протокол ([steam://rungameid/2923300/](steam://rungameid/2923300/))
2. Ждёт 1 минуту
3. Сносит процесс игры banana.exe
4. Ждёт 3 часа

повтор...

# Как использовать?

## Если установлен .NET 8.0
Скачай репозиторий.
Распакуй архив в любую папку.
Выполни эту команду в командой строке из той папки:
```cmd
dotnet run
```

## Если .NET НЕ установлен
Скачай релизную версию [здесь](https://github.com/DimucaTheDev/FermaBananov/releases)
